# Requirements Spec · 诺曼底碉堡

## 1. 概述
固定机位 2D 像素射击防守游戏。玩家操控碉堡上的艾森豪威尔,抵御从海面涌出、逼近防线的生化士兵无尽波次。

## 2. 功能需求
| ID | 需求 | 类型 | 验收 |
|---|---|---|---|
| FR-1 | 标题屏 → 点击/空格开始 | UI | 显示标题、操作说明、历史最高分 |
| FR-2 | 鼠标瞄准 + 点击/按住开火 | UI | 子弹沿准星方向飞出,有枪口火光 + 弹壳 |
| FR-3 | 键盘 A/D 在碉堡上微调站位 | UI | 英雄在限定范围内左右移动 |
| FR-4 | 弹匣(8 发)+ R 换弹 + 打空自动换弹 | UI | 弹药 HUD 实时显示,换弹有冷却 |
| FR-5 | 敌人从海面涌出、上岸、逼近碉堡 | UI | 出水动画 + 行走摆动 + 血条 |
| FR-6 | 命中扣血 / 击杀计分 / 连击加成 | UI | 血花粒子、击杀爆开、combo 倍率 |
| FR-7 | 敌人逼近碉堡 → 扣碉堡血量 | UI | 碉堡完整度条下降、屏幕震动 |
| FR-8 | 无尽波次,数量/速度/血量递增 | UI | 波间提示 + 整备间隔 |
| FR-9 | 波间空投补给(弹药/修复) | UI | 降落伞 + 拾取效果 |
| FR-10 | 碉堡血空 → Game Over + 最高分 | UI | localStorage 持久化 best |
| FR-11 | 精英变异敌(更高血/更高分) | UI | 紫色发光 + 尖刺外观 |
| FR-12 | P 暂停 | UI | 暂停遮罩,再按继续 |

## 3. 非功能需求
- 单文件 `index.html`,无构建、无外部 JS/图片依赖(字体走 Google Fonts CDN,离线降级到等宽)
- 目标 60fps;dt 上限 0.05s 防大延迟跳变
- 无控制台错误(favicon 404 可忽略)
- 触屏兼容(touchstart/touchmove)

## 4. 项目类型
`ui_only`(纯前端,无持久化后端;localStorage 不构成 has_persistence 后端契约)
