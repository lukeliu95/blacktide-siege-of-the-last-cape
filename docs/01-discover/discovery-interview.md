# Discovery Interview · 诺曼底碉堡

## JTBD
当我想放松几分钟时,我想要一个**一打开就能玩、手感爽快的像素射击小游戏**,这样我能在熟悉的二战诺曼底主题里获得即时的"守住防线"的成就感。

## Goals(≥3)
1. **G1 即玩性** — 零安装零依赖,双击 HTML 即进游戏;3 秒内理解怎么玩。
2. **G2 爽快射击手感** — 瞄准跟手、枪口火光、弹壳抛出、命中血花、击杀爆开、屏幕震动。
3. **G3 渐进压力** — 波次越来越多越快,弹药管理 + 碉堡血量制造取舍张力。
4. **G4 主题沉浸** — 可辨识的艾森豪威尔 + 诺曼底海滩防御工事氛围。

## 非目标
- 不追求历史严谨(生化士兵为虚构设定);不做剧情;不做多关卡编辑器。

## 关键决策记录
| 议题 | 决策 | 理由 |
|---|---|---|
| 机位 | 固定机位波次防守 | 贴合"站碉堡打海里敌人"的原始画面 |
| 美术 | 程序化像素 | 单文件零资源,随处可跑 |
| 难度 | 无尽递增 | 计分驱动重玩 |
| 操作 | 鼠标瞄准 + 键盘移动 | 射击游戏最直觉映射 |
