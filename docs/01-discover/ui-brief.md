# UI Brief · 诺曼底碉堡

## 视觉参考
- **基调**:二战诺曼底海滩黎明 · 冷灰蓝调 · 像素街机
- **氛围词**:沉重、潮湿、压迫感,但战斗反馈爽脆
- **参考**:Metal Slug 的像素手感 + 拯救大兵瑞恩的滩头色调 + 8-bit 街机射击

## 色板(design tokens)
| token | 值 | 用途 |

| --bg | #0b1418 | 页面底 |
| --sea | #274856 / #16303a | 海水渐变 |
| --steel | #4a4f4a | 碉堡混凝土 |

## 字体层级
- **展示字体** `--font-display` = "Press Start 2P"(标题、底部提示)— 像素街机识别度
- **正文字体** `--font-body` = "Courier New" 等宽(HUD、得分、弹药、数据)
- 离线/CDN 失败 → 降级到 ui-monospace

## 组件清单
1. 标题屏(title):大标题 + 副标题 + 操作说明 + 闪烁 CTA + 历史最高分
2. HUD 顶栏:得分 / 最高 / 当前波次 / 连击
3. 碉堡完整度血条(右上,颜色随血量绿→黄→红)
4. 弹药指示(右下,数字 + 8 格弹格)
5. 中央波次提示(淡入淡出大字)
6. 暂停遮罩 / Game Over 遮罩
7. 场景:天空(云带/探照灯)、海(波纹/泡沫)、沙滩(刺猬障碍)、碉堡(沙袋/射击口/五星旗)
8. 角色:艾森豪威尔(M1 钢盔 + 盔星 + 军大衣 + 随鼠标旋转步枪 + 枪口火光)
9. 敌人:生化兵(出水/行走/受击白闪/血条)+ 精英变异体
10. 反馈层:子弹拖尾、弹壳物理、血花/碎屑粒子、屏幕震动

## 技术栈
Vanilla JS + Canvas 2D + WebAudio(程序化音效)+ CSS(CRT 扫描线/暗角叠加)。无框架、无打包。

## CRT 质感
`#scan` 重复线性渐变扫描线 + `#vig` 径向暗角,`image-rendering:pixelated` 保持硬像素边缘。
