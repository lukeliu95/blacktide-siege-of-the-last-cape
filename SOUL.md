# SOUL · 诺曼底碉堡:艾森豪威尔的最后防线

## Core Truths
- 这是一个**即玩、爽快、有压力**的固定机位像素射击游戏,不是历史模拟器。
- 每一次开火都要有反馈:火光、弹壳、血花、震动、声音——爽感来自这些细节的叠加。
- 难度靠"波次递增 + 弹药管理 + 碉堡血量"三件套制造取舍,而不是靠不公平的判定。

## Boundaries
- 单文件 `index.html`,永不引入构建步骤或外部美术资源。
- 生化士兵是虚构设定;不渲染血腥写实,只用像素化血花表达打击感。
- 不加入账号、联机、付费、广告。

## Vibe & Tone
- 视觉:诺曼底黎明的冷灰蓝 + 街机像素 + CRT 扫描线。
- 听觉:程序化 8-bit 音效,枪声脆、命中闷、击杀低沉、空投清亮。
- 情绪:孤胆守线的悲壮感 × 街机得分的轻快感。

## Capabilities
- 标题 / 战斗 / 暂停 / Game Over 完整状态机。
- 鼠标瞄准射击、弹匣换弹、键盘移动。
- 无尽波次、精英敌、连击计分、空投补给、碉堡血量、localStorage 最高分。
- 桌面 + 触屏输入。
